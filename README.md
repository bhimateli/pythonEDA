#  Exploratory Data Analysis of Global Water Consumption

##  Project Overview

This project performs Exploratory Data Analysis (EDA) on a global water
consumption dataset to understand water usage patterns, environmental
impacts, and water scarcity trends across different countries and years.

The goal is to identify relationships between water consumption,
agricultural use, rainfall, groundwater depletion, and water scarcity.
These insights can help support sustainable water management and future
predictive modelling.

------------------------------------------------------------------------

##  Objectives

-   Analyze global water consumption patterns
-   Understand sector-wise water usage
-   Identify environmental factors affecting water scarcity
-   Explore correlations between rainfall, groundwater depletion, and
    water consumption
-   Generate insights useful for sustainability and resource planning

------------------------------------------------------------------------

##  Dataset Information

Source: Kaggle

Dataset Characteristics

-   Total Records: 3900
-   Scope: Multiple countries
-   Time Coverage: Multi-year dataset
-   Missing Values: None

### Key Features

-   Total Water Consumption
-   Agricultural Water Use
-   Industrial Water Use
-   Household Water Use
-   Rainfall Levels
-   Groundwater Depletion Rate
-   Water Scarcity Level

##  Data Preparation

Steps used for preparing the dataset:

1.  Imported dataset using Pandas
2.  Inspected dataset using df.info() and df.describe()
3.  Validated missing values and duplicates
4.  Standardized numerical data types
5.  Prepared clean dataset for analysis

------------------------------------------------------------------------

##  Exploratory Data Analysis

### Distribution Analysis

-   Histograms used to analyze variable distributions
-   Box plots used to detect outliers

### Correlation Analysis

Key findings:

-   Strong positive correlation between Total Water Consumption and
    Agricultural Water Use
-   Groundwater depletion strongly linked to water scarcity
-   Countries with lower rainfall rely more on groundwater

------------------------------------------------------------------------

##  Trend Analysis

The analysis shows a gradual increase in global water consumption over
time due to:

-   Population growth
-   Agricultural demand
-   Industrial expansion

------------------------------------------------------------------------

##  Key Insights

1.  Agriculture dominates global water consumption.
2.  Water scarcity results from multiple environmental factors.
3.  Groundwater depletion poses a major sustainability risk.
4.  Water usage patterns vary significantly across regions.
